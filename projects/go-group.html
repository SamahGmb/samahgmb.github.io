Go Group

the project

Project carried out in a group of 4 people as part of the Wagon Bordeaux training. The goal was to make an AirBnb clone.

features

As an organizer, you can

  • Find an activity by name, by location or by date
  • Add an event
  • Delete an event

As a participant, you can

  • Find an activity by name, by location or by date
  • Join an event
  • See the list of all activities you joined
