Go Group

the project

A simple app to meet people around you through activities. You can create an event or join one.

This project was carried out in a group of 4 people as part of the Wagon Bordeaux training. The goal was to make an AirBnb clone.

View online *

* This web app uses the free version of Heroku. The time for the first load may be a bit long. The server goes to sleep every hour.

features

As a user, you can

  • You can register or connect to your member area
  • Search for an activity by name, by location or by date, using the search bar
  • Edit your profile

As an organizer, you can

  • Add and publish an event
  • See the list of all the activities you organize
  • Delete an event

As a participant, you can

  • Participate in an event
  • See the list of all the activities in which you participate
